# Extract Todo Skill

이 스킬은 생성된 피드백 리포트를 분석하여 사용자가 별도의 태스크로 진행할 수 있는 실행 가능 항목(Action Items)을 추출하고, 이를 관리하기 쉬운 Todo 리스트 마크다운 파일로 저장합니다.

## 실행 절차 (Execution Steps)

1. **최신 피드백 읽기**: `.agents/vault/records/` 하위의 가장 최신 마크다운 기록 및 피드백 파일을 읽어옵니다.
   - 사용자가 특정 파일을 지정했다면 해당 파일을 우선적으로 읽습니다.
2. **Action Item 추출**: 파일 내용 중 `추가 학습 제안 (Next Steps)` 및 `즉각적 실행 유도 (Actionable Next Step)` 섹션을 중점적으로 스캔하여, 실천 가능한 형태의 내용(예: 특정 리팩토링 실습, 개념 스터디 등)을 추출합니다.
3. **Todo 형식 변환**: 추출된 항목들을 `[ ]` 형태의 체크리스트를 포함한 Markdown 리스트로 가공합니다.
4. **Todo 파일 저장/업데이트**:
   - 워크스페이스 최상단 디렉토리 혹은 지정된 경로(예: `.agents/vault/todos.md` 또는 `docs/todos.md`)에 있는 기존 Todo 파일을 찾아 내용을 덧붙입니다(Append).
   - 파일이 없다면 새로 생성합니다.
   - 항목을 추가할 때는 어떤 작업 기록 파일에서 유래했는지 맥락(Context Link 또는 출처)을 반드시 함께 기재합니다.
5. **결과 알림**: 추가된 항목의 갯수와 저장된 위치를 사용자에게 요약하여 보고합니다.
