# PR 메시지 템플릿

> 아래 템플릿을 기반으로 PR 메시지를 작성하세요.
> `[...]` 부분을 실제 내용으로 교체하고, 해당 없는 섹션은 삭제합니다.

---

## 제목

```
[타입] [핵심 변경 내용을 한 줄로 요약]
```

**예시:**

- `[feat] 구독 카드 UI 개선 및 실시간 사용량 반영`
- `[fix] 무료 도구 전체 선택 시 체크박스 오동작 수정`

---

## 본문

### 작업 목적

> 이번 PR이 왜 필요한지 1~2줄로 간략히 설명합니다.

[이번 작업의 배경이나 해결하려는 문제점 등을 작성합니다.]

### 주요 변경 사항

> 정말 중요한 핵심만 bullet point로 요약합니다. 불필요한 파일 나열이나 사소한 수정 사항은 제외하세요.

- [핵심 변경 사항 1]
- [핵심 변경 사항 2]

### 주요 확인 사항 (Optional)

> 리뷰어가 중점적으로 봐야 할 부분이나 특이사항, 관련 이슈 번호 등이 있다면 간략히 기재합니다. 해당 사항이 없다면 이 섹션을 삭제하세요.

- [리뷰 포인트 혹은 특이사항]
- 관련 이슈: #[이슈 번호]
